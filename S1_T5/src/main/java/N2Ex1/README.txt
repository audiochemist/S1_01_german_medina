Open a terminal or command prompt in your operating system.

1 - Type "cd" + the path to the content root. For example:
cd "directory_path"

2 - Compile the files:
javac Setting.java DirectoryPropertyWriter.java App.java

3 - Execute the app:
java App

The Program will create a txt file according to the parameters of the setting.properties file.
