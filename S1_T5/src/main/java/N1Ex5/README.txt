Open a terminal or command prompt in your operating system.

1 - Type "cd" + the path to the content root. For example:
cd "directory_path"

2 - Compile the files:
javac User.java Serializer.java App.java

3 - Execute the app providing 2 arguments: one number + the directory path as command-line arguments:

(Numeric Arguments)
1 Show directory list
2 Read a txt file
3 Serialize a User object
4 Deserialize that same object

Example in order to execute the program:
java App 3 "directory_path"
